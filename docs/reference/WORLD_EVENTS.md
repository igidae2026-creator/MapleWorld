# World Events

World dynamics stay within one server instance but still vary the play session.

- daily and weekly event activation through `scripts/world_event_system.lua`
- regional modifiers exposed on map snapshots
- rare mob cadence through spawn rarity
- dynamic boss pressure and phased mechanics
- loot multipliers and boss spawn bonuses via active events
- invasion pressure carried into regional event state for higher-tension loops
- live-event activation exposed through the GM and admin surfaces
