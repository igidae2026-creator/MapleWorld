# Gameplay Variance Rules

The offline loop must preserve route choice, reward contrast, and memorable risk. It is not enough for gameplay to be balanced. It must also remain textured.

## Level-Band Variance Floors

Each meaningful level band must preserve all of the following:

- one safe map
- one alternative map
- one high-risk high-reward map

These route roles do not need identical throughput. They must remain meaningfully different.

## Reward Variance Floors

Rare drops must remain.

- uncommon, rare, elite, and boss layers must not collapse into mostly common output
- chase rewards must continue to exist as visible spikes

Boss reward variance must remain.

- bosses must not all feel like the same expected-value package
- miniboss and boss reward identity must remain differentiated

## Early Loop Texture Floors

The early game must preserve:

- first combat
- first quest
- first equipment upgrade
- first potion purchase
- first map transition
- first party opportunity
- first miniboss encounter

The loop may become cleaner, but it may not become generic.

## Veto Rule

If a patch improves stability but reduces variance, reject patch.

This includes:

- collapsing map-role separation
- removing rare reward spikes
- smoothing early-game route identity into a single optimal path
- normalizing locked canon assets into generic substitutes
